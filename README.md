# Brandon Travel Pages

A Jekyll site for GitHub Pages that publishes structured travel intelligence reference pages under the Brandon Travel brand.

## Included data pages

1. Languages by Country
2. Time Zones by Country
3. Major International Airports
4. Airport Codes Reference
5. Airline Alliances
6. Digital Nomad Visas by Country
7. Schengen Area Countries
8. Transit Hubs by Region
9. Visa-Free Travel by Passport
10. Global Mobility Hubs

## Local development

```bash
bundle install
bundle exec jekyll serve
```

## Notes

- Designed to visually match the Brandon Travel tools site.
- Uses a flat page structure suitable for GitHub Pages.
- Starter tables are intentionally lightweight and can be expanded over time.
